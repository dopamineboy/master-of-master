# 평민 → 마스터 육성 게임 — 기획 요약

**목표**: 평민으로 시작해서 행동·성장을 반복하며 **마스터**까지 올라가는 육성 게임.

---

## 1. 컨셉

- **시작**: 이름만 정한 **평민** 한 명. (직업 선택은 선택 사항: 어부/농부/광부/장사꾼 등)
- **끝**: **마스터** 칭호 달성 = “이 게임에서의 성장 완료” 느낌. (그 이후에도 계속 플레이 가능)
- **중간**: 레벨 + **칭호**로 구간을 나눠서 “어디쯤 성장했는지”가 보이게.

---

## 2. 성장 구간 (칭호 예시)

| 구간 | 칭호 예시 | 느낌 |
|------|-----------|------|
| 1 | **평민** | 시작. 아무것도 없는 상태 |
| 2 | **견습** / **후보생** | 뭔가 배우기 시작 |
| 3 | **초급** / **이수생** | 기초를 넘어섬 |
| 4 | **숙련** / **경력직** | 꽤 해봤다 |
| 5 | **전문가** / **장인** | 잘한다 |
| 6 | **명인** / **대가** | 그 분야의 최상위 |
| 7 | **마스터** | 성장의 정점 |

- 레벨과 칭호를 **함께** 올려도 되고, “레벨 N 구간 = 칭호 O”처럼 **레벨만으로 칭호 결정**해도 됨. (구현 단순: 레벨 1~10 = 평민, 11~20 = 견습, … , 61+ = 마스터)
- **1차 구현**: 칭호는 “레벨 구간에 따른 이름 표시”만 해도 충분. 나중에 칭호별 보상·이벤트 추가 가능.

---

## 3. 핵심 루프 (무엇을 하면 성장하는가)

- **행동** (에너지 소모): 낚시 / 채집 / 채광 / 장사 중 하나 실행 → **아이템·골드·경험치** 획득.
- **성장**: 경험치 → **레벨업** → 최대 에너지·칭호 상승. 골드 → 장비 강화·구매 등.
- **정리**: 가방에서 아이템 확인 → 판매 또는 조합 → 골드/가공품으로 다시 행동에 투자.

**1차 구현 시 단순화 예**  
- 행동은 **낚시만** 넣어도 됨. (낚시 → 물고기 → 판매 → 골드 → 낚시대 강화 → 더 좋은 물고기 → 경험치/레벨)
- 이후 **채집·채광·장사**를 같은 패턴으로 추가.

---

## 4. 데이터에서 꼭 넣을 것

- **유저**: `name`, `level`, `exp`, `gold`, `energy`, `maxEnergy`, **`title`**(또는 칭호 = 레벨로 계산).
- **성장**: 레벨업 시 `maxEnergy` 증가, `title` 갱신 (레벨 구간표 참고).
- **진행**: “다음 칭호까지” 또는 “다음 레벨까지” 필요한 경험치/행동 횟수를 화면에 표시하면 “평민 → 마스터”가 체감됨.

---

## 5. 구현 순서 제안

1. **칭호·레벨 정의**  
   - 레벨 1~10 = 평민, 11~20 = 견습, … , 61+ = 마스터 같은 **구간표**를 코드/설정에 정의.
2. **백엔드**  
   - 유저 상태에 `level`, `exp`, `title`(또는 레벨로 계산) 반영. 행동 1회 시 exp 부여, exp 누적 시 레벨업.
3. **프론트**  
   - 상단에 **이름 / Lv / 현재 칭호(평민~마스터)** 노출. 경험치 바(선택)로 “다음 레벨까지” 표시.
4. **콘텐츠**  
   - 낚시만 먼저 완성한 뒤, 같은 방식으로 채집·채광·장사 추가.

이렇게 하면 “평민에서 마스터로 성장하는 육성 게임”이 뼈대부터 잡힌다.  
다음 단계에서는 **칭호 구간표 + 백엔드에 exp/level/title 반영**부터 코딩하면 된다.

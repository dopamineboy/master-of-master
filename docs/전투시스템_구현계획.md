# 전투·강화·먹이사슬·조합 시스템 구현 계획

## 1. 먹이사슬 (매달 체인지)
- **구현됨**: `battleSystem.js` - getFoodChain(), doesABeatB()
- farmer > miner > fisherman > merchant > farmer (기본, 월별 rotate)
- 먹이사슬 크리티컬: 25% (우위일 때 낮은 전투능력으로도 승리)

## 2. 전투능력 1~50단계
- **구현됨**: gameEngine - combatPower, getEnhanceProb()
- 강화 확률: 1 - level*0.025 (극악, 50단계≈0.001)

## 3. 전투 시스템
- **추가 필요**: handleGameCommand에 '전투'|'battle' 분기
- 하루 3회, 다른 직업 랜덤 매칭
- 승: 상대 골드 10% 획득 / 패: 내 골드 10% 차감 / 무승부: 에너지 -10

## 4. 강화석 시스템
- 강화석 조각 10개 → 강화석 1개 조합
- 강화석 1개 소비하여 전투능력 강화 시도 (확률)

## 5. 크로스 조합
- 농부 레어5 + 어부 일반10 → 슈퍼푸드 (에너지+50)
- 어부 레어5 + 농부 일반10 → 이직계약서
- 광부 레어5 + 어부 일반10 → 강화석 조각

## 6. UI
- 내정보: 레벨 → 전투능력
- 채널 하단: 전투 버튼, 전투 결과창 (승/패 내역)
- 전투 클릭 시 애니메이션·이펙트

- battle, enhanceCombat, craftCross, useSuperFood 응답 필드 추가

# 노가다 MASTER — 플레이어 가이드

> 게임을 처음 플레이하는 분을 위한 **조작법·화면 설명·팁**입니다.

---

## 1. 시작하기

### 1.1 캐릭터 생성

1. **아이디** 입력 (2~12자 한글·영문·숫자) → **중복확인**
2. **비밀번호** 입력 (8자 이내 한글·영문·숫자)
3. **캐릭터 생성** 버튼 클릭
4. 직업은 **랜덤**으로 정해집니다 (농부 / 낚시꾼 / 광부 / 장사꾼)

오른쪽 상단 **로그인** 버튼으로 기존 계정 로그인 가능

직업은 한 번 정해지면 바꿀 수 없습니다. 이직계약서를 사용하면 직업을 바꿀 수 있습니다.

### 1.2 로그인

이미 만든 캐릭터가 있다면 **로그인** 버튼을 눌러 아이디·비밀번호로 접속하세요.

> ⚠️ 계정 데이터는 서버 메모리에만 저장됩니다. 서버를 재시작하면 모든 계정이 초기화됩니다.

---

## 2. 화면 구성

### 2.1 채널 화면 (메인)

| 영역 | 설명 |
|------|------|
| **상단 메시지** | "○○ 채널에 입장하셨습니다." |
| **에너지 게이지** | 현재 에너지 / 최대 에너지. 행동하면 소모, 매일 오전 12:00(자정) 리셋 |
| **버튼** | 내 정보, 가방, 채팅, 알림 |
| **지도** | 밭 · 광장 · 장터 이동 |

### 2.2 지도 섹터

- **밭** → 직업별 행동(채집/낚시/채광/장사) 수행
- **광장** → 철수 vs 영희 배팅
- **장터** → 아이템 판매·구매

---

## 3. 밭 (핵심 행동)

### 3.1 행동 버튼

직업에 따라 다음 버튼이 나타납니다.

| 직업 | 버튼 | 대기 중 표시 |
|------|------|--------------|
| 농부 | 채집하기 | 채집중입니다 |
| 낚시꾼 | 낚시하기 | 낚시중입니다 |
| 광부 | 채광하기 | 채광중입니다 |
| 장사꾼 | 장사하기 | 장사중입니다 |

- 버튼 클릭 → **에너지 2** 소모 → 일정 시간 대기 → 아이템·골드 획득
- 에너지가 부족하면 버튼이 비활성화됩니다.

### 3.2 획득물

- **일반** / **레어** / **유니크** 등급
- 레어·유니크는 드랍 확률이 낮지만 골드가 높습니다.
- 직업별 칭호(초보~마스터)는 활동 횟수가 누적되면 달성됩니다.

---

## 4. 가방

- **탭**: 농부 아이템, 낚시꾼 아이템, 광부 아이템, 장사꾼 아이템, 계약서, 상자, 소비 아이템
- 아이템 이름과 개수 표시
- **전부 판매**: 해당 탭 아이템을 한 번에 골드로 변환 (선택 가능)

---

## 5. 장터

### 5.1 일반 판매

- 기본 가격으로 아이템 판매 (직업별 드랍 아이템)

### 5.2 상인조합 풀

- 농부·낚시꾼·광부: 드랍 아이템을 **판매**하여 풀에 적립
- 장사꾼: 풀에 있는 아이템을 **구매** (다른 직업이 판매한 것)

### 5.3 상인판매 (내 판매 등록)

- **장사꾼만** 상인조합에서 산 아이템을 일정 이윤 범위 내에서 재등록 가능
- 모든 직업이 **구매** 가능

---

## 6. 광장 (배팅)

- **철수 vs 영희** 승부 (30분마다)
- **배팅**: 30~100G, 철수 승 / 영희 승 / 무승부 중 선택
- 배당: 2배(철수/영희), 6배(무승부)
- 경기 결과는 실시간으로 갱신됩니다.

---

## 7. 내 정보

- **아이디**, **레벨**, **경험치**, **골드**, **에너지**, **칭호**
- 직업별 칭호 (초보○○ → 숙련○○ → 마스터○○)
- **전투능력** (전투 승률에 영향)

---

## 8. 전투

- 하루 **3회** 참가 가능
- 다른 직업과 랜덤 매칭
- **먹이사슬**: farmer > miner > fisherman > merchant > farmer (월별 변경)
- **결과**: 승리 시 상대 골드 10% 획득 / 패배 시 내 골드 10% 차감 / 무승부 시 에너지 -10

---

## 9. 강화·조합

### 9.1 전투능력 강화

- **강화석** 소비 → 전투능력 1 상승 시도 (확률)
- 강화석: 강화석 조각 10개 → 강화석 1개 (조합)

### 9.3 계약서 조합

- 일반계약서 10장 → 고급계약서 1장 (장터에서 일반계약서 구매 후 조합)
- 고급→전문→명인→마스터 순으로 조합 가능

### 9.4 크로스 조합

- 농부 레어 5 + 어부 일반 10 → 슈퍼푸드 (에너지 +50)
- 어부 레어 5 + 농부 일반 10 → 이직계약서
- 광부 레어 5 + 어부 일반 10 → 강화석 조각

---

## 10. 팁

1. **에너지**는 매일 오전 12:00(자정)에 리셋되므로, 매일 조금씩 플레이하는 것이 좋습니다.
2. **장사꾼**은 상인조합에서 저렴하게 구매한 뒤, 이윤 범위 내에서 재판매할 수 있습니다.
3. **레어·유니크** 아이템은 크로스 조합 재료로도 쓰이므로, 필요한 만큼 모아 두세요.
4. **광장 배팅**은 운 요소가 크지만, 적당한 금액으로 도전해 보세요.
5. **전투**는 먹이사슬 우위일 때 유리합니다. 직업 관계를 참고하세요.

---

## 11. 문제 해결

| 문제 | 확인 사항 |
|------|-----------|
| 버튼이 비활성화됨 | 에너지 2 이상 확인 |
| 장터에서 구매/판매 안 됨 | 골드·소지 아이템 확인, 직업별 권한(상인조합 등) 확인 |
| 전투 참가 불가 | 하루 3회 제한 확인 |
| 칭호가 안 올라감 | 계약서 보유·조합 여부 확인 (고급→마스터 단계) |

추가 문의는 개발팀 또는 커뮤니티를 이용해 주세요.
